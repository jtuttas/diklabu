Filter für Ergebnisse einer Umfrage

Umfragen können in der Haupt- und Vergleichsgruppe nach Klassen gefiltert werden. Geben Sie hier die Klassenbezeichnung ein, so erhalten Sie die Umfrageergebnisse der Klasse. Mehrere Klassen können durch ";" getrennt werden. Das "%" Zeichen dient als Wildcard.

Beispiele

  • FISI14A;FISI14B Es werden die Ergebnisse der Klassen FISI14A und FISI14B angezeigt.
  • FIAE% Es werden die Ergebnisse aller FIAE Klassen angezeigt.
  • SYK14%;IFK14% Es werden die Ergebnisse der Klassen SYK14 und IFK14 angezeigt.
  • % Es werden die alle Ergebnisse der Umfrage angezeigt.
